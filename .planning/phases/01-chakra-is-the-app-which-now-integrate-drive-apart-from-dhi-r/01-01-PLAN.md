## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host runtime environment -> core capability service | Untrusted host PATH/process environment influences dependency detection results. |
| Capability service -> startup gate | Incorrect translation can over-block or under-block application startup. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-02 | D | startupOrchestratorService | mitigate | Add tests for blocked/non-blocked transitions and exact missing-dependency messaging. |
| T-01-03 | I | diagnostics surface | mitigate | Return dependency names/status only; avoid leaking sensitive environment data in messages. |
